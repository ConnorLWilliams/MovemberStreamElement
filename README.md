# MovemberStreamElement
Plugin to create a custom widget for a friend streaming for Movember

To Use:

1. Clone / fork this repo into one that you can personally access.

2. Edit the url in the config file to your Movember link.

3. Make an account on Render.com

4. Make a web service and link it to your Github repo or git repository

5. Deploy the webservice and note the url that gets deployed

6. take the example widget javascript and html and make a custom widget in Stream Elements.

7. edit the url in the javascript to your Render.com webservice.

8. test and deploy
